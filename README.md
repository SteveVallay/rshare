rshare
=====

这是我第一个 rails web 程序。这个web 应用旨在帮助用户上传和分享和下载文档，不需要任何积分。

Ruby version : 1.9.3
Rails version : 4.0.0
OS: Linux


安装：

```
 git clone git@github.com:SteveVallay/rshare.git
 cd rshare/app
 bundle install
```
开启server:

```
rails s
```

打开[http://localhost:3000][1] 即可看到界面。
